To run the code:

1.Generate the project using CMake
2.Build the project called ass2
3.Run it

Basic Requirements:

1.Render a cube.
2.Implement basic camera controls using the mouse and keyboard.
3.Transform the cube to clip space using a ModelViewProjection matrix in the vertex shader.
4.Shade the cube with a single point light 

Advanced Requirements:

1.Render a second cube that orbits around the first
2.Render the cubes with textures.("fungus.dds","uvmap.DDS")
3.Implement loading and rendering of two Wavefront obj models from a file. ("mycube.obj","cube.obj")
